# Dossier des photos

Ce dossier contient les photos de l'équipe médicale de l'Hôpital Mame Abdoul Aziz Sy Dabakh.

## Structure des fichiers

### Photos du personnel médical :

- **`directeur.jpg`** - Photo du directeur de l'hôpital
  - Format recommandé : JPG ou PNG
  - Dimensions recommandées : 400x400 pixels (carré)
  - Taille maximale : 2 MB

- **`specialiste-1.jpg`** - Photo du premier médecin spécialiste
- **`specialiste-2.jpg`** - Photo du deuxième médecin spécialiste
- **`specialiste-3.jpg`** - Photo du troisième médecin spécialiste
- ... (ajoutez autant de photos que nécessaire)

### Format des images :

- **Format** : JPG ou PNG
- **Dimensions** : 400x400 pixels (format carré recommandé)
- **Taille** : Maximum 2 MB par image
- **Qualité** : Haute résolution pour un affichage net

### Instructions :

1. Placez vos photos dans ce dossier `images/`
2. Nommez les fichiers selon les conventions ci-dessus
3. Les photos doivent être au format carré (1:1) pour un meilleur rendu
4. Assurez-vous que les photos sont bien cadrées et professionnelles
5. Si une photo n'existe pas, une image par défaut avec une icône sera affichée

### Note technique :

Si une photo n'est pas trouvée, le site affichera automatiquement une icône de remplacement. Vous pouvez ajouter vos photos à tout moment sans modifier le code HTML.
